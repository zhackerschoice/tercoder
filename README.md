# tercoder
Encode and Decode using your terminal <br>
supports hex , base64 and rot13 Encodings <br>
works only in Linux <br>
Tested on Kalilinux 2021.1 <br>
# prerequirements <br>
python3 <br>
# installation <br>
git clone https://github.com/zhackerschoice/tercoder <br>
cd tercoder <br>
python3 main.py <br>
